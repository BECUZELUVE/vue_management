人员管理系统
==================================

**1. 简介**
----------------------------------
一款基于vue2.0的人员管理系统，样式使用iviewui框架。
数据库语言：mongoose
后台接口语言：node.js

实现对数据库中的人员信息的增加，编辑，删除，以及模糊查询。

**2. 使用：**
------------------------------
        数据导入数据库中
        数据库开机 mongod --dbpath 数据的存储路径
        后端运行：npm install
                node app.js
        前端运行：npm install
                npm run dev
